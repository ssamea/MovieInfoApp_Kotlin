# MovieInfoApp_Kotlin
코틀린으로 개발한 코로나 방콕을 위한 영화 추천 정보 어플리케이션
이 어플의 주요 기능으로는 영화 포스터 제공, 영화 이름 검색, 유튜브 API를 연동한 영화 예고편 제공기능, 평점 기능등이 있음.
